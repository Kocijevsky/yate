Changelog
=========

0.0.40
------

  * Изменен синтаксис атрибутов с динамическим именем.
    Было:

        @{ .name } = 42
        @{ "data-{ .name }" } = 24

    Стало:

        @{ .name } = 42
        @data-{ .name } = 24

    Т.е. статические куски имени (строковые литералы) не нужно заключать в кавычки.
